play2VR

Javascript全局对象, 网页需先接入play2VR.js（请参考play2VR控制台->视频管理->视频编缉->接入代码）

播放器方法

.init( container , key )

初始化播放器, 播放器会在容器位置出现并替换容器内容
-- container (必填)播放器的容器元素
-- key (必填)视频key（请参考play2VR控制台->视频管理->视频上传）

.dispose()

销毁播放器,并恢复容器原来的内容

.getDiv()

返回容器元素

.getMedia()

返回封装后的媒体对象

.on( eventName , callback )

监听播放器事件（注意：在.dispose()之后所有绑定的事件监听即失效，新视频初始化后需要重新绑定）
-- eventName (必填)事件名称
-- callback (必填)回调函数

.off( eventName , callback )

取消监听播放器事件
-- eventName (必填)事件名称
-- callback (必填)回调函数

.toggleVR( vr )

切换VR模式
-- vr (可选)true/false, 不填自动切换状态

.toggleGyro( gyro )

切换启用陀螺仪传感器
-- gyro (可选)true/false, 不填自动切换状态

.toggleFullscreen( fullscreen )

切换全屏播放
-- fullscreen (可选)true/false, 不填自动切换状态

.toggleMessage( message )

切换播放器自带的消息动画（加载/播放/暂停）
-- message (可选)true/false, 不填自动切换状态

.play()

播放视频

.pause()

暂停视频

视频交互方法

.addCuePoint( name , time )

添加视频时间事件, 自定义视频时间事件可以使用.on()来监听
-- name (必填)事件名称
-- time (必填)事件时间, 视频播放的进度条秒数

.removeCuePoint( name )

删除视频时间事件
-- name (可选)事件名称, 不填删除所有视频时间事件

.addHotSpot( name , options )

添加视频热点, 可交互的视频热点可以使用.on()来监听
-- name (必填)热点名称
-- options (必填)热点参数对象, {image: 图片网址或画布, pos: 三维向量数组, interactive: 是否可交互}

.removeHotSpot( name )

删除视频热点
-- name (必填)热点名称

.getHotSpotByName( name )

返回名字相符的视频热点对象, 可以进一步显示或隐藏热点对象
-- name (必填)热点名称

视频热点对象方法

.show()

显示视频热点

.hide()

隐藏视频热点

.updateImage()

更新视频热点贴图,画布绘制后调用

播放器事件

load 播放器初始化完成事件,监听到此事件之后方可调用播放器方法

dispose 播放器销毁事件

loadskin 播放器皮肤初始化完成事件

play 播放事件,事件参数是currentTime

pause 暂停事件,事件参数是currentTime

toggleVR 切换VR事件

toggleGyro 切换启用陀螺仪事件

toggleFullscreen 切换全屏事件

toggleMessage 切换播放器自带的消息动画（加载/播放/暂停）事件

orientationchange 横屏,竖屏播放事件, 事件参数是屏幕方向: landscape/portrait
